## 📥 Getting Started with NexusCore

NexusCore is designed for immediate deployment on Windows 10 (build 1909+) and Windows 11. The installation process is optimized to take less than 90 seconds on modern hardware.

- **OS:** Windows 10/11 (64-bit)
- **RAM:** 4 GB minimum (8 GB recommended for HD Streaming module)

1. Obtain the activation token from the NexusCore distribution portal.
2. Run the installer and select "Pro Edition" from the feature matrix.

4. Reboot your system to complete shell integration.

- Module selection (choose which features to enable)

## 🛠️ Technical Architecture

NexusCore operates on a **microservices architecture** using lightweight Windows services and shell extensions:

```
NexusCore Service Layer
├── CoreDaemon (always running)
│   ├── Module Loader
│   ├── Update Scheduler
│   └── Telemetry Filter
├── User Interface (WPF + WinUI 3)
│   ├── Dashboard
│   ├── Configuration Editor
│   └── Module Manager
└── Extension Points
    ├── Shell Context Menu (regsvr)
    ├── Clipboard Monitor
    ├── Network Proxy (for StreamLift)
    └── Accessibility Bridge
```

Each module runs in its own **isolated AppContainer** with minimal permissions — only the HD streaming module requires network access, and only the emoji module accesses the clipboard. This granular permission model ensures that a compromised module cannot affect the rest of the system.

### Performance Metrics
- **Cold start:** 2.3 seconds to dashboard readiness
- **Memory footprint:** 45 MB baseline, 120 MB with all modules active
- **CPU usage:** < 0.3% idle, up to 5% during HD enhancement
- **Disk writes:** Minimal (configuration changes only, < 1 MB/hour)
